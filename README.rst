=======
Mancify
=======

SSH with a Manc flair!

Try it out!
-----------
Example::

    >>> from mancify.translator import translate
    >>> translate('This is bad!')
    'This iz pants!'

Requirements
------------

* NLTK 2.0.4+

See `requirements.txt` for details.

Testing
-------
To test in Python 2.7+::

    python -m unittest discover
